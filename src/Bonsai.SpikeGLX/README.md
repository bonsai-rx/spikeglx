## About

`Bonsai.SpikeGLX` provides operators for interfacing with the [SpikeGLX](https://billkarsh.github.io/SpikeGLX/) recording software for [Neuropixels](https://www.neuropixels.org/) probes.

## How to Use

To use `Bonsai.SpikeGLX` for visual reactive programming, please install this package using the [Bonsai package manager](https://bonsai-rx.org/docs/articles/packages.html).

## Documentation

For additional documentation and examples, refer to the [official Bonsai documentation](https://bonsai-rx.org/spikeglx/).
